# 🏏 IPL Data Analysis Dashboard (2008–2025)

## 📌 Project Overview
This project is an interactive **IPL (Indian Premier League) Dashboard** built using **Power BI**.  
It analyzes match data from **2008 to 2025** to provide insights into team performance, player statistics, and season highlights.

The goal of this project is to transform raw cricket data into **meaningful insights through visualization and storytelling**.

---

## 🚀 Features

- 📅 **Season-wise Analysis (2008–2025)**
- 🏆 **Champion & Runner-Up Identification**
- 📊 **Key Season Metrics**
  - Total Matches
  - Total Teams
  - Total Venues
  - Total 4s & 6s
  - Centuries & Half-Centuries
- 🟠 **Orange Cap (Most Runs)**
- 🟣 **Purple Cap (Most Wickets)**
- 💥 **Top Batting Stats (Most 4s & 6s)**
- 📋 **Points Table (Team Performance)**
- 🎛️ **Interactive Slicer for Dynamic Filtering**
